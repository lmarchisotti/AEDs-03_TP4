Para executar este programa é necessario que compile e execute após.

Para compilar: com o terminal aberto na mesma pasta que os arquivos do programa estão digite:
make 

Para executar: com o terminal aberto na mesma pasta que o passo anterior foi executado:
./tp3 -i input (sendo "input" o arquivo de entrada desejada para a execução)


obs: Nesta mesma pasta contém um arquivo de entrada chamado input, caso nao deseje executar com esta entrada, ignore o arquivo ou apague o mesmo.
